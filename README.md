# Project-Mirage
Project-Mirage, is an enterprise cloud security solution that uses strategic deception to neutralize threats. By deploying a high-fidelity matrix of decoy assets and bait credentials, it misdirects attackers into isolated traps, triggering immediate, zero-false-positive alerts before core data can be breached.

## How to Run Locally

To get this project running on your local machine, you only need to run a few commands in your terminal:

1. **Clone the repo**
   ```bash
   git clone https://github.com/arham7006/Project-Mirage.git
   cd Project-Mirage
   ```

2. **Install dependencies**
   *(Note: The actual `node_modules` are not pushed to GitHub to keep the repo clean and fast. This command downloads them automatically based on the `package.json`.)*
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Click the `http://localhost:5173/` link that appears in your terminal.
